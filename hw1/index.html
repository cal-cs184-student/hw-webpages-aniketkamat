Overview

In this assignment, I implemented a rasterizer that can help display SVG files. It tries to solve the core computer graphics problem of displaying 3D figures on a 2D screen, so it implements tools like antialiasing and pixel/level sampling to make the 2D image look more realistic and appear to have gradient/texture features. It helped me learn how to use simpler techniques such as triangle rendering and matrix multiplication to solve complex display problems. One interesting thing I learned was that the supersampling method is unexpectedly good for how simple it is to implement compared to many other methods. When I was doing the extra credit for Task 2, I experimented with a few different filtering techniques and saw that most others either produced worse artifacts or more uneven coloring.
