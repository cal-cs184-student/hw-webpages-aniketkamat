Section I: Bezier Curves and Surfaces

Part 1: Bezier curves with 1D de Casteljau subdivision

De Casteljau’s algorithm is an algorithm that uses linear interpolation to evaluate a Bezier curve given its control points. Given n control points and a parameter t in between 0 and 1, De Casteljau's algorithm computes n-1 linearly interpolated values between the points using the formula (1-t)*pi+ t*pi+1 (eg. ith interpolated value is between control points i and i+1). This is one step of the algorithm. For the next step, the interpolated values from the last step become the new control points and a set of n-2 values (since there are n-1 control points) are calculated with interpolated. This can be repeated until the final step where 1 interpolated value is given by the final 2 control points. I implemented this using the steps above in three separate functions. The first two functions implemented a single step of the algorithm by looping through the control points and storing the interpolated values. The third function implemented the entire algorithm by performing a similar loop and continuing until there was only one value (the final interpolated one).
I then moved around the control points to create a slightly different Bezier Curve to see how t affects the parameter t affects the execution of the algorithm

Part 2: Bezier surfaces with separable 1D de Casteljau

De Casteljau’s algorithm can be extended to Bezier surfaces by looping/interpolating across one direction first and then interpolating across those values for the other direction.
